# Deployment Recommendation Framework for Efficient Stable Diffusion Turbo Inference on Simulated Edge Devices

## Overview

This project investigates the deployment of Stable Diffusion Turbo under simulated edge-device constraints. We evaluate different deployment configurations—including schedulers, inference steps, image resolutions, and runtimes—to analyze their impact on latency, memory usage, and overall inference performance.

Based on the experimental results, we develop a lightweight Deployment Recommendation Framework that recommends suitable deployment configurations for different resource-constrained environments.

## Objectives

- Implement Stable Diffusion Turbo using Hugging Face Diffusers.
- Evaluate inference performance under simulated edge constraints.
- Compare multiple deployment configurations.
- Measure latency, memory usage, CPU/GPU utilization, and image quality.
- Develop a recommendation framework for selecting efficient deployment settings.
- Document experimental results for research publication.
